# Blackout

A lightweight GNOME Shell extension that displays a black screen after a period of user inactivity and automatically enables power saving mode.

The black screen helps reduce power consumption and screen wear while keeping the current session active. Any mouse movement or keyboard input instantly restores the normal desktop view.

Features
Displays a black screen after a configurable idle timeout
Automatically enables power saving mode when activated
Instantly restores the desktop on user activity
Lightweight and resource-friendly
Seamlessly integrates with GNOME Shell

Ideal for users who want a simple energy-saving solution without locking the screen or suspending the system.
