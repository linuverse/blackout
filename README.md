# Blackout

A lightweight GNOME Shell extension that displays a black screen after a period of user inactivity.

The black screen helps reduce power consumption on OLED displays and minimizes the risk of burn-in, while keeping the current session active. On LCD/LED displays, it can still help avoid displaying static content for extended periods. Any mouse movement or keyboard input instantly restores the normal desktop view.

## Features
- Displays a black screen after a configurable idle timeout
- Helps reduce power consumption on OLED displays
- Helps minimize the risk of OLED burn-in
- Instantly restores the desktop on user activity
- Lightweight and resource-friendly
- Seamlessly integrates with GNOME Shell

Ideal for users who want a simple way to blank the screen without locking the session or suspending the system.
